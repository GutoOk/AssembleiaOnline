# **App Name**: Assembleia - Mensa Brasil

## Core Features:

- Google Authentication: Secure login exclusively via Google accounts with @mensa.org.br domain validation.
- Role-Based Access: Differentiate user access levels (member vs. administrator).
- Assembly Management: Administrators can create and manage assembly details (title, description, date).
- YouTube Live Stream Integration: Embed a YouTube live stream link within each assembly for members to view.
- Polling System: Administrators can create polls with custom or default options ('Sim', 'Não', 'Abstenção') and set time limits for voting.
- Transparent Voting: Open vote system where all members can see who voted for what, with a final vote report showing totals and individual votes.
- Public Speaking Queue with Zoom Integration: Members can request to speak and are added to a queue, where administrators can manage the queue and provide a Zoom link to the first in line.

## Style Guidelines:

- Primary color: Deep green (#00642E) to convey trust and professionalism, in reference to the serious conduct of meetings.
- Background color: Light green (#F0F2F5), complementing the primary color and ensuring readability.
- Accent color: Soft green (#7CB342) to subtly draw attention to calls to action and key information, sitting close to primary color in hue but creating some visual separation via differences in brightness and saturation.
- Body and headline font: 'Inter', a grotesque sans-serif for a clean and modern appearance suitable for digital interfaces.
- Use a set of professional and easily recognizable icons, consistently styled to represent different actions and categories.
- Maintain a clean, organized layout with clear sections for assemblies, polls, and the speaking queue.
- Incorporate subtle transitions and feedback animations to enhance user interaction.
